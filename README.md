# Лабораторная работа: Анализ образа жизни студентов

## Описание задачи

Датасет `student_lifestyle_100k.csv` содержит 100 000 записей о студентах. Каждая запись описывает поведенческие характеристики: часы учёбы, сна, физической активности, время в соцсетях, уровень стресса, а также успеваемость (CGPA) и наличие депрессии.

Решаются две задачи:
- **Регрессия** — предсказать CGPA
- **Классификация** — предсказать депрессию

## Структура ноутбука

### Разведочный анализ (EDA)
Построены гистограммы распределений, тепловая карта корреляций, ящики с усами для выявления выбросов. Проверены пропуски и дубликаты. Обнаружен дисбаланс целевой переменной: депрессия встречается в 10% случаев.

### Предобработка
- Удалёны идентификаторы Student_ID, Age, Department
- Переименованы столбцы для удобства работы
- Дублирующихся строк не было выявлено, соответсвенно, нечего удалять
- Выбросы обработаны через ограничение по межквартильному размаху (capping)
- Выполнен андерсемплинг: количество здоровых студентов сокращено до 10 000 для баланса классов

### Линейная регрессия (CGPA)

Использованы признаки: Study, Sleep, Media, Activity, Stress.
Данные разделены в пропорции 60/20/20, применён StandardScaler (обучен на train, преобразованы val и test).

**Итоговые метрики:** R² = 0.044, MAE = 0.435.
**Вывод:** Корреляция признаков с успеваемостью крайне слабая, модель не пригодна для предсказаний.

### Логистическая регрессия (Depression)

Сравнивались три модели: LogisticRegression, SGDClassifier, CatBoost.

**Лучший результат** показала логистическая регрессия на сбалансированных данных:
- Precision = 0.547
- Recall = 0.500
- F1 = 0.522
- 
### Проверка переобучения
Сравнение метрик на валидации и тесте показало расхождения менее 2% для двух моделей — переобучение отсутствует. CatBoost - переобучался в двух случаях.

### Влияние балансировки классов

| Условие | F1 (LogisticRegression) |
|---------|------------------------|
| Без балансировки (90/10) | 0.188 |
| С балансировкой (50/50) | 0.522 |

Балансировка классов дала значительный прирост качества.
